1. Which of the following is the time complexity of accessing an element in an array (random access)?
A) O(1)
B) O(log n)
C) O(n)
D) O(n^2)
Answer:A

2. In a binary search tree (BST), for any node, the value of all nodes in its left subtree is:
A) Greater than the node's value
B) Smaller than the node's value
C) Equal to the node's value
D) Not related to the node's value
Answer:B

3. Which of the following data structures is used for implementing function calls in most programming languages?
A) Queue
B) Stack
C) Linked List
D) Hash Table
Answer:B

4. What is the time complexity of searching for an element in a balanced binary search tree (BST)?
A) O(1)
B) O(log n)
C) O(n)
D) O(n log n)
Answer:B

5. Which algorithm is commonly used for finding the shortest path in a graph with non-negative edge weights?
A) Quick Sort
B) Merge Sort
C) Dijkstra's Algorithm
D) Floyd-Warshall Algorithm
Answer:C
